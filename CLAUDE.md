## Overview

This is a Slay the Spire 2 mod that replaces epoch art on the timeline. It compiles a C# DLL and packages image assets into a Godot `.pck` resource pack.

## Build

### Prerequisites

Copy `local.props.example` to `local.props` and set:
- `<STS2GamePath>` ŌĆö path to your STS2 installation directory
- `<GodotExePath>` ŌĆö path to Godot 4.5.1 Mono executable

### Build command

```bash
PATH="$HOME/.dotnet:$PATH" ~/.dotnet/dotnet build EpochReplacer.csproj
```

The `PATH` prefix is required on Linux: Godot's PCK export step shells out to `dotnet`, which must be on `PATH` even though the MSBuild step itself uses the full `~/.dotnet/dotnet` path.

The build automatically:
1. Generates `pack/project.godot` and `pack/mod_manifest.json`
2. Compiles C# to `EpochReplacer.dll`
3. Invokes Godot to export assets as `EpochReplacer.pck`
4. Writes final artifacts to `dist/EpochReplacer/`

### Output

Three files are deployed to `dist/EpochReplacer/`:
- `EpochReplacer.dll` ŌĆö compiled mod assembly
- `EpochReplacer.pck` ŌĆö Godot resource pack containing the replacement texture
- `mod_manifest.json` ŌĆö mod metadata for the STS2 mod manager

## Architecture

### Mod loading

The STS2 mod loader calls `Initialize()` (marked with `[ModInitializer]`) at startup. The method loads the replacement texture from `res://images/custom_epoch.png` inside the `.pck`, with a fallback to load the PNG directly from the assembly directory.

### Texture replacement

The mod uses Harmony postfix patches to replace textures at the exact moment each node sets them, with zero per-frame overhead. The libgcc_s preload is required on Linux before any `harmony.Patch()` call (see `roadblocks.md` in the knowledgebase).

Patches hook the methods that assign `_portrait.Texture` / `_blurPortrait.Texture`:
- `NEpochSlot.SetState()` ŌĆö covers all three slot states; `%Portrait` for Complete/Obtained, `%BlurPortrait` for NotObtained (which uses a SubViewport blur effect on `%Portrait`)
- `NEpochCard.Init()` ŌĆö `_Ready()` does not set the portrait; Init does
- `NEpochInspectScreen.Open()` ŌĆö texture is assigned before the first `await`, so the postfix fires after it's set
- `NEpochInspectScreen.OpenViaPaginator()` ŌĆö private method called when paging between epochs while the screen is open

### Asset pipeline

- `pack/images/custom_epoch.png` ŌĆö the replacement texture (edit this to change the art)
- `pack/export_presets.cfg` ŌĆö Godot export config for Windows and Linux `.pck` export
- `pack/project.godot` ŌĆö generated by MSBuild, do not edit manually
